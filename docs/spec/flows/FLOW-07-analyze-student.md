# FLOW-07: Analyze Assessments (Student)

**Actor:** Student
**Traces to:** REQ-EVAL-03, REQ-EVAL-04

---

## Preconditions

- Student has completed at least one challenge run.

## Steps

1. Student navigates to their assessment history.
2. System displays a list of completed challenge runs with qualitative grades (A–C). *(REQ-EVAL-04)*
3. Student selects a challenge run to view details.
4. System shows per-phase feedback and improvement advice: separate grades and feedback for Framing, Judging (across cycles), and Steering (across cycles). *(REQ-EVAL-03, REQ-EVAL-04)*
5. Student reviews grades and feedback across all three rubric dimensions.

## Postconditions

- Student has reviewed their performance and feedback.
