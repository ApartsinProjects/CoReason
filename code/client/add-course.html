Build your subject tree by adding topics and sub-topics. Drag to reorder, click + to nest.

    YAML Subject Tree Format

    Each node has a name and optional children array:

    - name: "Sorting & Searching"
  children:
    - name: "Comparison-based Sorting"
    - name: "Hash-based Searching"
    - name: "Binary Search Variants"
- name: "Graph Algorithms"
  children:
    - name: "Shortest Paths"
    - name: "Minimum Spanning Trees"
- name: "Dynamic Programming"

    Alternatively, use simple indented lists (dashes with indentation to indicate nesting):

    - Sorting & Searching
  - Comparison-based Sorting
  - Hash-based Searching
- Graph Algorithms
  - Shortest Paths
  - Minimum Spanning Trees
