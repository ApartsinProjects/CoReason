Challenge Evaluation Rubric

Framing
A: All gaps identified, justified assumptions B: Most gaps found, reasonable assumptions C: Partial identification, weak justification
Judging
A: All errors detected, logical analysis B: Most errors found, some analysis gaps C: Few errors caught, surface-level review
Steering
A: Specific, actionable, converges to solution B: Mostly specific, partial convergence C: Vague directions, slow convergence
