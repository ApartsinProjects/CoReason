The CoReasoning Process

This diagram shows the three-phase co-reasoning flow you follow in every challenge.

Phase 1 - Framing

You receive an ill-defined problem and must refine it into a well-defined task by adding problem definitions and AI task instructions.

Phase 2A - Judging

You review the AI-generated solution and identify real issues versus non-issues. Your judgments test your critical evaluation skills.

Phase 2B - Steering

You write specific correction commands to fix the issues you found. The AI regenerates an improved solution, and the cycle can repeat.

Each phase is graded independently (A / B / C). Judging and Steering repeat in cycles until completion.
